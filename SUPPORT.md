# Support

- **Bug reports and feature requests:** open a GitHub issue using the
  templates under `.github/ISSUE_TEMPLATE/`.
- **Questions and discussion:** {{project.links.support_chat}}
- **Documentation:** see the `docs/` directory, starting with
  `docs/architecture/overview.md`.
- **Security issues:** see `SECURITY.md` - please do not open a public
  issue for security reports.
